Features

User and product management

Users can have specific role on each product. Users and roles can be added, updated or removed by a user with super-admin profile or a user with a manager role.

Release

A product can have many releases defined by a release date.

Baseline

A baseline is a reference point in the development life cycle of the product. A reference such as a software revision control system tag or label can be added.
